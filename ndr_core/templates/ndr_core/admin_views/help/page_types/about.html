About Page

Most project web sites feature an "About out the Project" page. You can create such a page automatically by selecting this page type when creating a page.

How To Create An "About" page

  • Go to Style & Colors > Manage Page Images and upload images from your project members to the image group "People Images". Choose the name of the person as image title and their role in the project as caption.
  • Go To Manage Pages > Create New Page to create a new page.
  • Choose "About Us Page" as type.
  • Fill in your project description in the "Template text" field.
  • Fill out the necessary fields. For example: "About Us" as page title "About" as navigation label and "about" as view name.
  • Create the page.

Features of an About Page

Once created, an about page shows your template text and a list of people involved in your project. They are displayed as card elements featuring their name and their role in the project.
