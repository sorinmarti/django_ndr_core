API Ninjas API

API Ninjas is a service which provides access to a large number of simple APIs. You'll need an API Key to use the service. You get 50K requests for free, after that you need to subscribe to a premium plan. With the free plan there is no support or commercial use.

In the context of NDR Core, API Ninjas APIs help you build test environments or demonstrate NDR Cores ability to get data from somewhere and display it in a pretty form.
