Ndr core Help

This is a help page

Basic Setup

In order to access this administration interface, you executed the 'init_ndr_core' command in your django installation. With this, your installation has been initialized and a number of Settings have been set. You can change these on the Settings Page.

Change the basic look and feel

NDR core comes in different styles. Your ndr/base.html file extends a certain style. You can change this to a different style or create your own.

Template Pages

Template pages serve static content (such as a project description ar curated studies). When creating a Template Page you choose a view_name. This is the path where this page will be served and the name of the template file created in your installation. Edit the file to provide the static content.

Search Pages

Search Pages come in three types: Simple search, Custom search and Combined search. A simple search consists of a single catch-all search field which searches all fields. The custom search consists of a number of search fields which you can configure. The combined search shows two tabs, one containing the simple search, the other one a custom search. The search forms need an API to connect to. This is the host where the search results are served from. The following rules/restrictions apply:

  • A custom search needs a Search Configuration. A search configuration is made up of different Search Fields. You'll have to create the search fields first and then add them to a search configuration.
  • Search Fields, Search Configurations and API Configurations can be reused on different pages.
  • A simple search cannot be further configured. It is one field searching in all the data.

List Pages

A list page can display a list of data which can be filtered.

Contact Form

A Contact Form is a form where visitors can send you a message. The message will be sent to an e-mail address of your choice. The following rules/restrictions apply:

  • You can have only one contact form in your installation
  • Go to the Settings Page to configure your E-Mail server to send them.
